# Cilika Home — лендинг

Статический одностраничный сайт строительной компании **Cilika Home** (частные дома в Элисте и Республике Калмыкия): приветственный экран, блоки услуг и проектов, калькулятор ориентировочной стоимости, FAQ, форма заявки, переключение светлой/тёмной темы.

## Структура репозитория

| Путь | Назначение |
|------|----------------|
| `index.html` | Разметка и скрипты; в `<head>` — ранние скрипты темы и приветствия |
| `assets/css/main.css` | Все стили страницы (светлая/тёмная тема через переменные и `data-theme`) |
| `README.md` | Описание проекта и запуск |
| `.gitignore` | Служебные и временные файлы |

## Запуск локально

Откройте `index.html` в браузере двойным щелчком или через простой HTTP-сервер (так корректнее работают некоторые сценарии):

```bash
cd путь/к/репозиторию
python3 -m http.server 8080
```

Сайт: `http://localhost:8080/`

## Функциональность

- **Приветствие** — показывается при первом открытии сайта в вкладке браузера; повторно в той же сессии не появляется (`sessionStorage`).
- **Тема** — кнопка в правом нижнем углу, предпочтение сохраняется в `localStorage`.
- **Калькулятор** — примерная стоимость по площади, этажности, комплектации и материалу.
- **Форма заявки** — клиентская проверка и демонстрационное сообщение (без отправки на сервер).
- **FAQ** — раскрывающиеся ответы.
- **Адаптив** — шапка с бургер-меню на узких экранах.

## Git-репозиторий

Если папка ещё не под версионным контролем, в корне проекта выполните:

```bash
git init
git add .
git commit -m "Initial commit: лендинг Cilika Home"
```

Дальше можно подключить удалённый репозиторий на GitHub/GitLab и пушить ветку `main`.

## Публикация

Подойдёт любой хостинг статики: GitHub Pages, Netlify, Cloudflare Pages, VPS с nginx. Достаточно выложить каталог с `index.html` в корень сайта.
